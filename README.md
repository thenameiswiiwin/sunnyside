## Getting Started

### Clone Repo

```
git clone https://github.com/thenameiswiiwin/sunnyside.git
```

### Change Directory

```
cd sunnyside
```

### Install Dependencies

```
npm install
```

### Scripts

##### Run StyleLint

```
npm run lint:css
```

##### Run ESLint

```
npm run lint:js
```

##### Run ESLint & StyleLint

```
npm run lint
```

##### Prettier

```
npm run format
```
