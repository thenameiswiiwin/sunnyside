## Getting Started

```bash
# Clone Repo
$ git clone https://github.com/thenameiswiiwin/sunnyside.git

# Install Dependencies
$ npm install
```

## Scripts

```bash
# Run StyleLint
$ npm run lint:css

# Run ESLint
$ npm run lint:js

# Run ESLint & StyleLint
$ npm run lint

# Prettier
$ npm run format

```
